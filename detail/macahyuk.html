MacaHyuK – Digital Library

Github

Deskripsi

Tech Stack: PHP, MySQL

MacaHyuK adalah aplikasi web perpustakaan digital yang memungkinkan pengguna untuk mengakses koleksi buku, melakukan peminjaman, serta mengelola data perpustakaan melalui peran User dan Admin. Aplikasi ini dikembangkan menggunakan PHP dan MySQL.
