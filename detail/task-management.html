Deskripsi

Tech Stack: Laravel, Alpine.js, MySQL

Aplikasi Task Management ini dirancang untuk membantu pengguna mengatur aktivitas secara lebih terstruktur dan efisien. Dengan antarmuka yang bersih dan intuitif, pengguna dapat membuat, mengedit, mengelompokkan, serta mengurutkan task dengan mudah. Sistem ini dilengkapi fitur kategori, prioritas, subtask, pencarian, pengurutan cerdas, dan kemampuan drag-and-drop untuk memindahkan task sesuai kebutuhan.

Aplikasi juga menyediakan halaman khusus untuk melihat semua task yang telah diselesaikan, lengkap dengan pengelompokan berdasarkan tanggal selesai. Selain itu, terdapat tampilan kalender interaktif yang memungkinkan pengguna melihat task berdasarkan tanggal dan menyunting task dengan cepat dari panel samping.

Halaman Dashboard

  • Task Management Panel: Daftar task rapi lengkap dengan checkbox dan priority bubble.
  • Edit Task Panel: Menampilkan detail task aktif.
  • Priority System: Indikator prioritas 1–5.
  • Category Filter: Memfilter task berdasarkan kategori.
  • Manual Sorting: Drag & drop mode.
  • Subtask Manager: Tambah/edit/hapus subtask.
  • Search Tasks: Pencarian real-time.
  • Manage Categories: Modal CRUD kategori.
  • Sort Options: Berdasarkan due date, creation date, atau manual.

Halaman Completed Tasks

  • Riwayat Tugas Selesai: Tampilan rapi dan minimalis.
  • Grouping Tanggal: Otomatis berdasarkan completed date.
  • Uncheck to Restore: Mengembalikan status task.
  • Clean UI: Warna lembut membedakan dari task aktif.

Halaman Calendar

  • Interactive Calendar: Navigasi bulan dan hari.
  • Dot Indicators: Penanda tanggal yang memiliki task.
  • Day-Based Task Listing: Klik tanggal → tampilkan task.
  • Edit Task Panel: Panel kanan menjadi editor task.
  • Component Reuse: UI konsisten antar halaman.
